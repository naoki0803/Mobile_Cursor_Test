詳細なカリキュラム構成

Phase 1: 基礎編（約15-20時間）

.NET基礎（5-7時間）
  • .NETの歴史と進化の理解
  • CLRとBCLの仕組み
  • 開発環境のセットアップ（Visual Studio/VS Code）
  • プロジェクト構造の理解
期待される成果:

.NETプラットフォームの全体像を理解し、開発環境を整備できる

C#基本文法（10-13時間）
  • 変数、データ型、演算子
  • 制御構造（条件分岐、ループ）
  • オブジェクト指向プログラミング
  • LINQ、非同期プログラミング
期待される成果:

C#でクラスベースのアプリケーションを作成できる

Phase 2: Web開発編（約25-30時間）

ASP.NET Core（8-10時間）
  • HTTPパイプラインの理解
  • ミドルウェアの仕組み
  • 依存性注入（DI）
  • 設定とロギング
期待される成果:

基本的なWebアプリケーションを構築できる

MVCパターン（10-12時間）
  • Model-View-Controllerの役割
  • ルーティングとアクションメソッド
  • Razorビューエンジン
  • フォーム処理とバリデーション
期待される成果:

MVCパターンでWebページを作成できる

Web API（7-8時間）
  • RESTful APIの設計
  • JSONデータの処理
  • APIバージョニング
  • Swaggerドキュメント
期待される成果:

RESTful APIを設計・実装できる

Phase 3: データ層編（約20-25時間）

Entity Framework Core（12-15時間）
  • Code FirstとDatabase First
  • マイグレーション管理
  • リレーションシップの定義
  • クエリ最適化とパフォーマンス
期待される成果:

EF Coreでデータベース操作を効率的に実装できる

データベース設計（8-10時間）
  • 正規化とパフォーマンス
  • インデックス戦略
  • トランザクション管理
  • セキュリティ考慮事項
期待される成果:

スケーラブルなデータベース設計ができる

Phase 4: 実践プロジェクト編（約40-50時間）

認証・認可（10-12時間）
  • ASP.NET Core Identity
  • JWT認証
  • ロールベースアクセス制御
  • OAuth2.0連携
期待される成果:

セキュアな認証システムを実装できる

ECサイト構築（20-25時間）
  • 商品カタログ管理
  • ショッピングカート実装
  • 決済システム統合
  • 在庫管理と注文処理
期待される成果:

完全なECサイトを開発できる

デプロイメント（10-13時間）
  • Azure/AWS展開
  • CI/CDパイプライン
  • 監視とロギング
  • パフォーマンス最適化
期待される成果:

本番環境へのデプロイと運用ができる

カリキュラム修了時の総合的な成果
  • フルスタックのECサイトを独力で構築できる
  • エンタープライズレベルのコード品質を維持できる
  • セキュリティベストプラクティスを実装できる
  • チーム開発に必要なGit/GitHub操作ができる
  • トラブルシューティングと問題解決ができる
  • 新しい技術を自主的に学習し続けられる

段階的な開発アプローチ

Step 1: 基本構造（Week 1-2）
  • プロジェクトセットアップ
  • データベース設計
  • 基本的なモデル作成
  • 認証システムの実装
成果物: ユーザー登録・ログインができる基本アプリ
Step 2: 商品管理（Week 3-4）
  • 商品モデルとリポジトリ
  • 商品一覧・詳細画面
  • カテゴリー機能
  • 検索・フィルター機能
成果物: 商品を閲覧・検索できるカタログサイト
Step 3: カート・注文（Week 5-6）
  • カートシステム実装
  • セッション管理
  • 注文処理フロー
  • 在庫管理連携
成果物: 商品をカートに入れて注文できるECサイト
Step 4: 完成・最適化（Week 7-8）
  • 決済システム統合
  • 管理画面の実装
  • パフォーマンス最適化
  • 本番環境へのデプロイ
成果物: 完全に機能するECサイト（ポートフォリオ作品）
